submission_date(pango_lin, mutations=None, location)

outbreak_data.submission_date(pango_lin, mutations=None, location=None, server='api.outbreak.info')

Returns the most recent submission date by location

Arguments:
pango_lin:

A string. (Required).

mutations:

(Optional). A string or list of strings. Comma separated list of mutations.

location:

(Optional). If not specified, return most recent date globally.

return:

A pandas dataframe.

Example usage:

df = outbreak_data.submission_date('xbb.1', location='USA')
print(df)
