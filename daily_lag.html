daily_lag(location=None)

outbreak_data.daily_lag(location=None, server='api.outbreak.info')

Return the daily lag between collection and submission dates by location

Arguments:
location_id:

(Optional). If not specified, return lag globally.

return:

A pandas dataframe.

Example usage:

df = outbreak_data.daily_lag('LUX')
print(df)
