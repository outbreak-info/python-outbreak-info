global_prevalence(pango_lin, mutations, cumulative)¶

outbreak_data.global_prevalence(pango_lin, mutations=None, cumulative=None, server='api.outbreak.info')¶

Returns the global daily prevalence of a PANGO lineage

Arguments:
param pangolin_lineage

(Required).

param mutations

(Somewhat optional). Comma separated list of mutations.

param cumulative

(Somewhat optional). If true returns the cumulative global prevalence since the first day of detection.

return

A pandas dataframe.

Example: Get global info on lineage ‘XBB’:

df = outbreak_data.global_prevalence('xbb')
print(df)
